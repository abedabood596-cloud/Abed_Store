موقع إلكتروني متكامل ومتجاوب مخصص لبيع الأجهزة والإلكترونيات الحديثة. تم بناؤه باستخدام لغة **PHP** وقواعد بيانات **MySQL** لتقديم تجربة تسوق سلسة وآمنة للمستخدمين، بالإضافة إلى لوحة تحكم شاملة لإدارة المتجر.

---

## 💡 فكرة المشروع (Project Idea)
تسهيل عملية التسوق الرقمي للمنتجات الإلكترونية (مثل الحواسيب المحمولة، الهواتف الذكية، والإكسسوارات) من خلال توفير منصة تفاعلية سريعة تتيح للمستخدمين استعراض المنتجات وإضافتها لسلة التسوق وإتمام الطلبات، وفي المقابل تتيح لمدير النظام (الأدمن) إدارة المنتجات والأقسام والطلبات والمستخدمين من مكان واحد.

---

## 📝 وصف المشروع (Project Description)
المشروع عبارة عن تطبيق ويب للتجارة الإلكترونية يربط بين واجهة المستخدمين (العملاء) وواجهة الإدارة (الأدمن):
- **قسم العملاء:** يسمح للزوار بتصفح المنتجات حسب الفئات، البحث عن منتج معين، إنشاء حسابات جديدة، تسجيل الدخول، إدارة سلة التسوق الخاصة بهم، وطلب المنتجات مع إمكانية متابعة حالة طلباتهم السابقة.
- **قسم الإدارة (لوحة التحكم):** يتيح للمشرفين مراقبة إحصائيات المتجر بشكل فوري (عدد المنتجات، الطلبات، العملاء)، وإضافة أو تعديل أو حذف (CRUD) المنتجات والفئات، بالإضافة إلى معالجة الطلبات وتحديث حالاتها وتصفح بيانات العملاء المسجلين.

---

## 🌟 المميزات الرئيسية (Key Features)

### 🛒 للعملاء (Customers):
- **الصفحة الرئيسية التفاعلية:** تحتوي على شريط إعلاني (Carousel) للمنتجات المميزة وبطاقات لأحدث المنتجات المضافة.
- **تصفح وتصنيف متقدم:** عرض المنتجات مع تفاصيلها والفرز بحسب الأقسام.
- **ميزة البحث السريع:** إمكانية البحث عن المنتجات بالاسم بسهولة.
- **سلة التسوق الديناميكية:** إضافة المنتجات، تعديل كمياتها، أو إزالتها بالكامل مع حساب تلقائي للمجموع الإجمالي.
- **نظام الطلبات:** إمكانية تقديم طلب جديد واستعراض الطلبات السابقة وتفاصيلها.
- **إدارة الحساب:** تسجيل حساب جديد وتأمين البيانات وتغيير الجلسات بأمان.

### 🛡️ للمشرفين (Admins):
- **لوحة معلومات (Dashboard):** عرض إحصائيات سريعة وملخصة حول نشاط المتجر.
- **إدارة المنتجات (Products Management):** إضافة منتجات جديدة (الاسم، الوصف، السعر، المخزون، الصورة، القسم) وتحديثها أو حذفها.
- **إدارة الأقسام (Categories Management):** إضافة وتعديل وحذف الأقسام المختلفة.
- **إدارة الطلبات (Orders Management):** مراجعة جميع طلبات العملاء وتغيير حالاتها (Pending, Processing, Completed, Cancelled).
- **إدارة المستخدمين (Users Management):** عرض قائمة بكافة الحسابات المسجلة في النظام لمعرفة العملاء.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **لغة البرمجة (Back-end):** PHP (Native PHP / PDO لشريحة البيانات لتجنب SQL Injection).
- **قواعد البيانات (Database):** MySQL.
- **الواجهات الأمامية (Front-end):**
  - HTML5 & CSS3 (Custom styling & gradients).
  - Bootstrap 5 (لتصميم متجاوب ومنسق).
  - Bootstrap Icons (للأيقونات التفاعلية).
  - خطوط Google Fonts (عائلة خطوط Inter).

## 🚀 طريقة تشغيل المشروع (How to Run)

لشغيل هذا المشروع محلياً على جهازك، يرجى اتباع الخطوات التالية:

### المتطلبات الأساسية:
- بيئة خادم محلي مثل **XAMPP** أو **WampServer** أو **Laragon**.
- إصدار **PHP 8.0** أو أحدث.
- خادم **MySQL**.

### خطوات التثبيت والتشغيل:
1. **تحميل المشروع:** قم بتنزيل أو نقل مجلد المشروع `Electronics_Store` إلى المجلد المخصص للمواقع في خادمك المحلي:
   - لبرنامج XAMPP: ضع المجلد داخل `C:\xampp\htdocs\`
2. **بدء تشغيل الخادم:** افتح لوحة تحكم الخادم المحلي (XAMPP Control Panel) وقم بتشغيل خادمي **Apache** و **MySQL**.
3. **تهيئة قاعدة البيانات:**
   - افتح المتصفح وتوجه إلى `http://localhost/phpmyadmin/`.
   - أنشئ قاعدة بيانات جديدة باسم `electronics_store`.
   - اضغط على تبويب **Import** (استيراد) واختر ملف `DataBase.sql` الموجود في الجذر الرئيسي للمشروع، ثم اضغط على **Go** لتنفيذ وإنشاء الجداول وإدخال البيانات التجريبية.
4. **التحقق من إعدادات الاتصال:**
   - افتح الملف [db.php](file:///c:/Users/hp/Desktop/Electronics_Store/config/db.php) وتأكد من أن بيانات الاتصال (اسم المستخدم وكلمة المرور وخادم قاعدة البيانات) تطابق إعدادات جهازك. البيانات الافتراضية المحددة هي:
     - الخادم: `localhost`
     - المستخدم: `root`
     - كلمة المرور: فارغة `""`
     - اسم قاعدة البيانات: `electronics_store`
5. **تصفح الموقع:**
   - افتح متصفحك واكتب الرابط التالي للدخول إلى المتجر:
     `http://localhost/Electronics_Store/views/Customer/index.php`
   - أو توجه مباشرة إلى صفحة تسجيل الدخول لبدء تسجيل الدخول للعميل أو المشرف:
     `http://localhost/Electronics_Store/views/Auth/login.php`
